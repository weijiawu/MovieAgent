MovieAgent: Automated Movie Generation via Multi-Agent CoT Planning

Weijia Wu , Zeyu Zhu , Mike Zheng Shou
Show Lab, National University of Singapore
Technical Report Code arXiv

MovieAgent: Convert your idea to a Movie!

Pipeline for MovieAgent

The Overall Pipeline for MovieAgent. The proposed framework employs a hierarchical CoT reasoning process with director, scene plan, and shot plan agents to automate long-form movie generation.

Task 1: Script Synopsis to Keyframe/StoryBoard Generation

With the hierarchical data MovieAgent provide, existing methods can partially reconstruct a coherent storyline, maintaining consistent character identities and relationships.
